AUDIO VISUALIZATION

Creative Tool / Data Visualization

Griffin Keller

Description

The aim of this tool is to visualize realtime audio in a room. The program will use the computer's input to record different types of sound, which will affect an oscilloscope type wave that runs throughout the middle of the screen. Sharp changes in audio will leave impressions of the line on the screen, so that after a while, an abstract image will be created that can be saved.

Motivation

I wish to create this tool as a way to give weight to sound. I've always been interested in audio visualizaiton, and this seems like the right outlet to explore it's possibilies.

Take Away

the tool will serve dual purposes. The first will be to craete visual accompanyment to music or performances. The second will be to save moments with abstract imagery - to remember the emotion and feeling associated with a memory rather than an image.

Interaction

The user will first begin the audio capture by pressing a button(there could be various options of visaul style). After that, the program will work for itself, and the only other interaction necissary will be to stop the recording. However, during the recording process, the user will have the option to click and create other visual imagery/markers.
